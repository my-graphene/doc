## 1. MyGChain简介

MyGChain是一条公有链，是MyGChain数据交易所的底层区块链，不仅支撑着MyGChain的高频数据交易，还支持第三方开发应用，在MyGChain上开发应用不仅可以得到链上支持，还可以获得MyGChain多维度数据的对接，可以做出非常落地于民生的有价值应用。

MyGChain主网已于2018年6月上线.
